## Description

This is the frontend for Weather Forecast & Traffic Cam Website. The backend instance from here `https://github.com/yiyao8932/data-gov-sg-be` is required to be up and running in order for the frontend to be working correctly.

The frontend consists of a date input and a time input for user to input desired date. The location listing, forecast and traffic images are populated after the date and time have been passed in.

## Tech stack and libraries used

React.js\
Next.js\
Tailwind\
Material Tailwind\
Axios\
Cypress for testing

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tests

```bash
npm run cypress:run
```

All the testing related stuff are under `./cypress` folder
